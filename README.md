# Olist E-Commerce Sales Analysis

## 📌 Project Overview
This project analyzes the Olist Brazilian E-Commerce dataset using SQL in Google BigQuery and Power BI. The objective is to identify sales trends, customer behavior, payment methods, and product performance through data analysis and interactive dashboards.

## 🎯 Business Questions
- How do order trends change over time?
- What is the most common order status?
- What is the average delivery time?
- How many orders were successfully delivered?
- Which product categories receive the highest number of orders?
- Which payment methods contribute the highest transaction value?
- What are the top-performing product categories?
- What insights can support business decision-making?

## 🛠️ Tools Used
- Google BigQuery (SQL)
- Power BI

## 📈 Key Insights
- Total Orders: **99,441**
- Unique Customers: **96,096**
- Total Revenue: **16.01M**
- Average Review Rating: **4.09**
- Credit Card is the dominant payment method based on transaction value.
- Bed Bath Table is the top-selling product category.
- Most orders were successfully delivered.
